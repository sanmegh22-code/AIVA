# AIVA Widget Specification

## Purpose

AIVA is an AI assistant integrated into the Inventory SaaS dashboard.

---

## Widget Placement

Right side of dashboard.

---

## Initial State

Title: AIVA Assistant

Placeholder:

Ask AIVA anything...

Suggested Questions:

- Show inventory summary
- Show low stock items
- Show dead stock
